# Basic Sitax Analyzer (2018)

Uses test/syntactic_grammar.in to build the syntatic table, and runs the analises in test/program.txt

Developed for INE5426 (Compiler Construction, UFSC)

## Dependencies

Basic Lexical Analyzer - used to generate lexical tokens from the input program

## Build
./auto.sh - compiles and builds syntacticanalyzer

## Run
./build/syntacticanalyzer
